# descargar_letras_deezer
```
Descarga las letras de un disco de Deezer en formato .txt

Usage: descargar_letras_deezer [OPTIONS]

Options:
  -u, --url <URL>          url directa del disco (separar por comas si se introducen varias)
  -d, --destino <DESTINO>  ruta completa al directorio en el que se guardarán los archivos resultantes [default: ]
  -m, --max <MAX>          número máximo de descargas simultáneas [default: 3]
  -h, --help               Print help
  -V, --version            Print version
  ```
